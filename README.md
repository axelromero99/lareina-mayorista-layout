## Starting 🚀

### Pre-requisites 📋
1. Clone or download the repository 

```bash
git clone https://github.com/axelromero99/lareina-mayorista-layout.git
```

2. Change the working directory

```bash
cd lareina-mayorista-layout
```

Open the **index.html**

🌟 You have all you need! 🌟
